# Stormy
Project 6 for the Team Treehouse Android Techdegree.

## Summary
Stormy is a simple weather app that uses the darksky.net API for weather data.

## Explanation
There are three tabs, a main summary, an hourly forecast, and a daily forecast for the next seven days. You can also convert between Fahrenheit and Celsius by clicking on the pink button in the bottom right of the screen.
